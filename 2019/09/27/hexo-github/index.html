使用Hexo搭建博客

发表于 2019-09-27 更新于 2019-09-28 分类于 博客

准备工作

Hexo是一个静态博客框架，它使用 Markdown 解析文章。
另外，托管也很方便，我用 Github Pages 来托管，其余的托管方式请自行搜索。

预先安装

  • Node.js (Node.js 版本需不低于 8.6，建议使用 Node.js 10.0 及以上版本)
  • Git

安装Hexo

建站

这里的 <folder> 后续将用 vblog 代替。

默认 vblog/source/ 目录下没有 categories, tags, about 子目录，需要单独进行添加。

NexT主题

默认主题不喜欢，我用的是 NexT 主题（Version: V7.4.0）。

配置

  • 站点配置文件: vblog/_config.yml
  • 主题配置文件: vblog/themes/{theme}/_config.yml

站点配置

站点相关的配置，以下列举出的配置有：主题、Site、搜索等。

主题
Site
URL

可以通过更改 permalink 来指定文章的url，只需要在文章的头部加入 id 属性。这样做的目的是：在文章标题为中文时，url 不会出现中文。

Category & Tag

配置分类和标签的映射，可以避免在分类或标签列表页面的 url 出现中文。
